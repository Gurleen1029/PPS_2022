## Program 34:Write a program using concatenate function.
```C
#include<stdio.h>
#include<string.h>
int main()
{
    char a[]="I am a student of";
    char b[]=" Guru Nanak Dev Engineering College";
    strcat(a,b);
    printf("%s",a);
    return 0;
}
```C
**Output:I am a student of Guru Nanak Dev Engineering College**
